# Documentation

These docs relate to how to contribute to the WanMask project itself.

You can find the latest version of WanMask on [our official website](https://wanmask.io/).

For help using WanMask, visit our [User Support Site](https://metamask.zendesk.com/hc/en-us).

For up to the minute news, follow our [Twitter](https://twitter.com/metamask_io) or [Medium](https://medium.com/metamask) pages.

To learn how to develop WanMask-compatible applications, visit our [Developer Docs](https://metamask.github.io/metamask-docs/).
